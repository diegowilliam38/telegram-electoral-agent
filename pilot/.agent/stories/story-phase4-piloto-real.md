# Story: Fase 4 — Piloto com Usuários Reais e Feedback

## Objetivo

Abrir o bot para um grupo seleto de servidores de outras zonas eleitorais, monitorar sua performance ao vivo e coletar feedback prático.

## Critérios de Aceite

- [ ] Divulgar o link do bot (`t.me/...`) para os participantes do piloto.
- [ ] Coletar métricas após 1 semana de uso (verificar arquivos de log por exceções).
- [ ] Registrar perguntas que o modelo falhou em responder ou respondeu incorretamente.
- [ ] Criar e atualizar o documento de "Análise do Piloto" com os feedbacks recebidos.
- [ ] Propor os primeiros refinamentos ou correção de RAG baseados no uso real.

## Dependências

- Conclusão da Fase 3.
